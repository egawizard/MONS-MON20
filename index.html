ℹ️ How It Works

MON-20 is the first inscription token standard on MONAD testnet, similar to BRC-20 on Bitcoin.


Minting Rules:

  • ✅ Each mint is exactly 1000 MONS
  • ✅ Total supply: 21,000,000 MONS
  • ✅ Uses 18 decimal format internally
  • ❌ Mints above 1000 are INVALID
  • ❌ Invalid mints don't count toward progress
